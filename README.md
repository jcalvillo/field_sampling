# Fieldsampling

Simple "quick-and-dirty" site for keeping track of field sampling schedule.

* ~Ruby 2.3, Rails 5.1.1~
* Dev DB: sqlite3 // Production DB: postgres
* Deployed to Heroku (db create, init, etc all @ Heroku)
* Dates *seeds* included
